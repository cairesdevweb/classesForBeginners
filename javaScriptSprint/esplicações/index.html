Operadores Matemáticos e Lógicos



Operadores lógicos 

              =         //   Sinal de igual Sozinho representa atribuição de valor
            7 > 6       // true (verdadeiro) — maior que
            7 < 6       // false (de falso) — menor que
            7 >= 6      // true (verdadeiro) — maior ou igual que
            7 <= 6      // false (de falso) — menor ou igual a
            7 == 6      // false (de falso) — igualdade
            7 != 6      // true (verdadeiro) — diferente
            7 === 6     // false (de falso) — identidade (igualdade estrita)
            7 !== 6     // true (verdadeiro) — não identidade (diferença estrita)
      
        
          
Lógica NOT, AND, e OR

          - NOT ! significa negação. Transforma verdadeiro em falso, e falso em verdadeiro.
          - AND && retornará true somente quando ambas as variáveis Booleanas forem verdadeiras.
          - OR || retornará true quando pelo menos uma das variáveis Booleanas for verdadeira.

Adicionando funções ao meus elementos

Declarando uma Função

A declaração de uma função envolve especificar seu nome, parâmetros (se houver) e o código a ser executado quando a função for chamada. Uma função é definida usando a palavra-chave function.

         
 
            function nomeDaFuncao(parametro1, parametro2) {
              // Código a ser executado quando a função é chamada
            }

nomeDaFuncao: Este é o nome da função, que você pode escolher conforme necessário. parametro1 e parametro2: Estes são os parâmetros da função, que são como variáveis que podem receber valores quando a função é chamada. O código dentro das chaves {} é o corpo da função, onde você coloca as instruções que deseja executar quando a função é chamada.

Chamando a Função

Depois de declarar uma função, você pode chamá-la em seu código para executar o código dentro dela. Para chamar uma função, você usa o nome da função seguido por parênteses (). Se a função aceitar parâmetros, você passa os valores dos parâmetros entre os parênteses. 

        
          nomeDaFuncao(valor1, valor2);
